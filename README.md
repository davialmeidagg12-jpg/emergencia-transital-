Emergência Transital
Aplicativo focado em auxiliar motoristas (novatos e idosos) em situações críticas e identificação de sinalização.
Funções Principais:
Alertas de Emergência: Instruções rápidas para acidentes, obras e animais.
Acessibilidade: Interface com botões grandes e alto contraste.
Educação Viária: Ajuda imediata para quem não conhece as placas.
